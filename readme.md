Desafios de Lógica de Programação em Java
--
Repositório criado para praticar e consolidar os conceitos fundamentais de programação aprendidos na DIO, incluindo variáveis, operadores, estruturas de decisão, laços de repetição e funções.

Este repositório contém dois projetos:
1. Classificador de Nível de Herói
2. Calculadora de Partidas Rankeadas

Os desafios propostos pela Digital Innovation One (DIO), com foco no fortalecimento da base lógica e na construção de um portfólio consistente no GitHub.


Entendendo o Desafio
--
O objetivo deste repositório é aplicar, na prática, os conceitos estudados, estruturando soluções próprias para cada desafio.

Aqui você encontrará:
1. Código-fonte dos projetos.
2. Organização clara da lógica implementada.
3. Estrutura preparada para evolução futura.
4. Possibilidade de expansão com novos desafios.
